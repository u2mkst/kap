# **App Name**: 클래스 허브

## Core Features:

- 사용자 회원가입 및 로그인: 사용자가 안전하게 새 계정을 생성하고 기존 계정으로 로그인할 수 있도록 지원합니다.
- 홈페이지: 학원의 일반 정보, 최신 공지사항, 추천 강좌 등을 표시하는 랜딩 페이지를 제공합니다.
- 내 정보 페이지: 사용자가 자신의 프로필 정보를 보고 업데이트하며, 등록한 강좌 목록을 확인할 수 있는 개인화된 공간입니다.
- 강좌 탐색 및 상세 정보: 개설된 강좌 목록을 확인하고, 각 강좌의 상세 내용(설명, 일정, 강사 등)을 조회할 수 있도록 합니다.
- AI 학습 도우미: 사용자가 입력한 키워드를 기반으로 학습 프롬프트, 요약 또는 일반적인 학습 관련 질문에 대한 답변을 생성하는 AI 도구입니다.

## Style Guidelines:

- Primary color: 학문적 신뢰감과 집중력을 연상시키는 전문적이고 차분한 파란색 (#4782CF). 시각적 안정감을 줍니다.
- Background color: 가볍고 쾌적한 느낌을 주기 위해 주 색상과 같은 계열의 매우 연하고 희미한 푸른 회색 (#F3F6FB).
- Accent color: 주요 동작 버튼이나 강조된 정보를 위한 신선하고 명확한 청록색 (#67C4DA). 주 색상과 조화를 이루면서도 시선을 끕니다.
- Headline 및 Body 폰트: 'Inter' (산세리프). 현대적이고 객관적인 느낌을 주며, 긴 텍스트 블록에서도 뛰어난 가독성을 제공하여 교육 콘텐츠에 적합합니다.
- 직관적이고 미니멀한 라인 아이콘 또는 플랫 아이콘을 사용하여 학업, 사용자 계정, 강좌 등 핵심 기능을 명확하게 표현합니다.
- 간결하고 정돈된 그리드 기반 레이아웃을 통해 콘텐츠 가독성을 극대화하고, 다양한 화면 크기에 맞춰 유연하게 반응하는 디자인을 적용합니다.
- 탐색 또는 인터랙션 시 부드럽고 미묘한 전환 애니메이션을 적용하여 사용자의 몰입감을 높이고 쾌적한 사용 경험을 제공합니다.
